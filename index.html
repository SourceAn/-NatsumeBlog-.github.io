​ 图片内显示我这里有两个主要分支，这是因为昨天用另一个佬的方法使用hexo生成了静态的博客，那个方法自己设置了配置为main。

​ 在关联远程仓库之前需要创建本地仓库，需要在项目目录中Shift右击打开Powershell

​ 输入git init 并执行

​ 结果是在目录下创建一个名为.git的文件夹

​ 之后继续在Powershell中执行 New-Item .gitignore

​ 然后佬说要添加过滤规则，过滤掉不需要上传的文件。

​

​ 关联远程仓库，输入代码 git remote add origin git@github.com:[githubID]/[repositoryName].git

​ 可能会需要输入账号密码，昨天在ssh以后我的这一步可以直接跳过。

​ 关联命令执行后，使用代码 git remote -v 查询是否完成。

​ 通过依次执行：

​ git add .

​ git commit -m "some description"

​ git push -u origin master

​ 就能将本地文件同步到远程。
